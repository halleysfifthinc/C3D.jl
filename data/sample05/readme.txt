MACSAMPLE.C3D is a sample C3D file from a older Motion Analysis
Corporation system.  Note the odd group and parameter names that
deviate from the C3D standard.
Contains analog data in signed format - SGI INT format.

NEWWALK.C3D is a later C3D file from a Motion Analysis 
Corporation system.  This file includes the new MANUFACTURER group
and conforms to the C3D format definition in all respects.
Contains point and analog data - DEC INT format.
